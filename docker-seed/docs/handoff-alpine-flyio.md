# Operator note

This folder is copied into some deployments as **workspace documentation** for end users.

**Hosting, API, container, and environment-variable details** belong in the main application repository (source tree), not in seeded `docs/` files that ship on user volumes.

If you are an **end user**, use **`docs/system.md`** and **`CYRUS.md`** in this workspace. If you are an **operator**, open the repo’s developer documentation there.
