# Writing style — workspace default

Use this file so agents (outreach, content, etc.) can mirror the owner's preferred tone once you edit it.

## Principles

- **Clear and direct** — say what you mean; avoid filler.  
- **Audience-aware** — match formality to the reader (investor vs. peer vs. support).  
- **Specific over vague** — concrete examples and numbers beat generic claims.  
- **Professional** — correct grammar and punctuation; avoid hype and exaggerated claims.
